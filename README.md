<!-- GETTING STARTED -->

### Prerequisites

- php >= 7.4

### Installation

```shell
composer require sofiakb/lumen-api-key
```

## Usage

```php
```

### Commands list
```shell
php artisan apikey:activate
php artisan apikey:deactivate
php artisan apikey:generate
php artisan apikey:list
php artisan apikey:retrieve
php artisan apikey:table
```
